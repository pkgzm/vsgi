Valum web micro-framework written in Vala

Documentation

Valum provides an extensive user documentation to get you started with the framework.

The framework is built upon libsoup and automated with the waf build system, so reading about all these is a good start.

The valadoc.org website provides api specification of most libraries available in Vala.

Examples

This is a real Vala web application running on libsoup built-in HTTP server and we got real examples!

Basics

  • Hello world!
  • Hello world! (with a trailing slash)
  • HTTP query
  • HTTP headers
  • cookies
  • redirection
  • not found
  • not found (thrown)
  • parse url-encoded data
  • gzip compression

Router features

  • all methods
  • GET and POST
  • route parameter
  • typed route parameter
  • custom typed route parameter (permutations of abc)
  • route using a regular expression
  • route using a matcher callback
  • route using a custom HTTP method
  • scoped routing
  • subrouter
  • next

Extras

  • lua scripting
  • CTPL templating
  • memcached (get value)
  • memcached (set value)
